CMCB

Champion Mains Clubs Bot

You're currently at the homepage of CMCBot - a bot that updates a bunch of tables in a bunch of subreddits so you can find League of legends club you're looking for. You can find some additional information about this bot below.

"Boop Bloop Beep Dee-Doop" - CMCBot, 2017.

Opensource

The bot is openly developed and hosted on github, so you can always see the work done and participate in development.

Always online

Since the bot is hosted on heroku, he is is ever vigilant and will update the club tables every {{ update_time }} on a 24/7 schedule.

Additional features

Since this is an automated solution, there is a lot of room for improvement. For example, adding last time club owner was in game to the table.
